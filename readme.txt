КАК УСТАНОВИТЬ?

1. Установить чистый Modx 2.8.4.
2. Установить pdotools и minishop2.
3. Заменить файлы minishop2 на соответствующие из этого репозитория.
4. Сделать статичными чанки tpl.msCart, tpl.msMiniCart, tpl.msProducts.row.
5. Сделать статичными сниппеты msMiniCart, msOrder.
6. Изменить системные настройки:
        ms2_frontend_message_css : [[+cssUrl]]web/lib/izitoast/iziToast.min.css
        ms2_frontend_message_js : [[+jsUrl]]web/lib/izitoast/iziToast.min.js
        ms2_frontend_message_js_settings: оставить пустой



ЧТО НОВОГО?

1. Библиотека jQuery больше не используется. Код переписан на чистый JS.
2. Для работы с DOM в качестве селекторов используются data-атрибуты с префиксом ms-. В будущем, наверное, будет возможность менять селекторы.
3. Объекту document добавлены два события ms_before_send и ms_after_send.
   Первое срабатывает до отправки данных на сервер, в нём эти данные можно изменить/дополнить/удалить/проверить.
   Второе срабатывает при получении ответа с сервера. Оба события можно отменить стандартным методом e.preventDefault().
4. Сохранена возможность добавлять callback-функции, но массивы для добавления изменены (написать что за массивы).
   Этот функционал рассчитан на то, чтобы не менять стандартное поведение, а дополнить его.
5. Код разбит на классы. На данный момент их 4:
        а) MiniShop содержит общие методы, является родителем для двух следующих классов;
        б) msCart класс обработчик корзины;
        в) msOrder класс обработчик заказа;
        г) msNotify класс для работы с уведомлениями, его можно расширить для подключения своей библиотеки уведомлений (примеры есть).
    Собственно остальные классы тоже можно расширить, если требуется серьёзно изменить логику работы.
6. Система задумывалась как модульная, поэтому классы вынесены в отдельные файлы, а в default.js происходит их подключение и инициализация.
7. Добавлена интерактивность миникорзине. Теперь она полностью синхронизирована с обычной корзиной в реальном времени.



ЧТО НУЖНО ПРОТЕСТИРОВАТЬ?

1. Работу событий и callback-функций. Нужно попытаться добавить какой-нибудь функционал, например, показ скидки или что-то ещё, дайте свободу вашей фантазии.
2. Работу миникорзины при разных вариантах расположения.
3. Работу карточки товара. Её я не изменял, поэтому нужно на примере листинга заставить работать и карточку (добавить атрибуты полю количество и кнопке).
4. Попытаться сделать то, что вы делали раньше используя новую реализацию JS. Улучшить юзабилити.
